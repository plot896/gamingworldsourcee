Can't play the game?

If the game screen keeps flashing, the screen is messy, or the screen is blank, it may be because your device does not support hardware acceleration

You can enterset up，Change renderer to CANVAS

If you only see "Unable to load Phaser", then I guess you are using a very old browser. My website does not support versions below IE 9, please use Chrome or Firefox instead

If it is stuck in "Loading assets...", it may be

  1. The network is not strong
  2. Your browser cannot decode the sound

Regardless of the reason, you can enterset up, Tick Disable audio
